```bash

# Build for macOS
```

## üéÆ How to Play

### Quick Start

1. **Launch GTNW**
2. **Select Era** - Choose any year from 1789-2025
3. **Select President** - Pick from 47 administrations
4. **Choose Difficulty** - Easy / Normal / Hard / Nightmare
5. **Start Game** - Begin your presidency

### Game Loop

**Your Turn:**
1. **Select target country** (green country picker)
2. **Choose action:**
   - Shadow President (132 options)
   - Nuclear Strike (1-50 warheads)
   - Declare War (conventional)
3. **Action executes automatically** - Turn ends
4. **AI countries respond** - Watch terminal
5. **Handle crises** - Make decisions when they appear
6. **Read messages** - Accept/Decline diplomatic communications
7. **Next turn begins**

### Strategy Tips

**Early Game:**
- Build alliances before conflicts start
- Invest in intelligence networks
- Strengthen economy through trade
- Avoid unnecessary wars

**Mid Game:**
- Manage DEFCON carefully (lower = more crises)
- Balance diplomacy with military strength
- Use covert ops for deniable actions
- Respond to crises thoughtfully

**Late Game:**
- Pursue victory condition actively
- Nuclear deterrence prevents attacks
- Coalition building isolates rivals
- Crisis management becomes critical

**Pro Tips:**
- AI opponents remember everything (betrayals, alliances)
- Advisors' advice reflects their personality (hawk vs dove)
- Each president's era affects available actions
- Historical crises teach real lessons
- Consistent behavior builds reputation
- "What If" simulator shows consequences before acting
